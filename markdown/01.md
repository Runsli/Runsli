这里是 **RunsLi**。一个运行在 Windows 指令集与 Android 开放生态下的极简记录点。

# 0x01 运行：开源、折腾与生产力 (The Runs)
我是一个深度 Windows 用户，也迷恋 Android 的自由度。

- **底层调教**：关于 Windows 效率优化、Android 玩机及各类跨平台软件的折腾心得。 
- **技术进阶**：记录从零开始的学习历程，把碎片化的知识整合成可运行的逻辑。 

# 0x02 滑行：感官、日常与节律 (The Slides)
在屏幕代码之外，我寻找那些不需要“配置”就能顺滑流动的时刻。 

- **学习足迹**：不仅是编程，还有对这个世界的各种认知更新。 
- **随笔记录**：那些不需要逻辑支撑的日常，以及偶然触动的感悟。 
- **数字断舍离**：在工具过载的时代，如何保持生活的轻盈。 

# 0x03 站点哲学
